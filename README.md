# Personal Brand Portfolio | NextJS
This is a [Next.js](https://nextjs.org/) personal brand portfolio project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

![Soren D. Blank](https://github.com/sorenblank/personal-brand-portfolio-nextjs/blob/main/public/og-image.png "I couldn't watch one piece lately")

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.
